# Banking Management System (Backend)

A Java-based banking application implementing admin-approved account creation.

## Tech Stack
- Java
- Servlets & JSP
- JDBC
- MySQL
- Apache Tomcat

## Features
- Customer submits account request
- Requests stored with PENDING status
- Admin login and dashboard
- Admin approves/rejects requests
- Account created only after approval

## Architecture
- MVC pattern
- Servlet for business logic
- DAO for database operations
- JSP for presentation

## Database
- account_request
- admin
- account

## How to Run
1. Import project into Eclipse
2. Configure MySQL schema
3. Deploy on Apache Tomcat
4. Access via browser
